# Netflix-Content-Analysis-Interactive-Tableau-Dashboard

Interactive Tableau dashboard analyzing trends across a 5,000+ title Netflix dataset — genre, content rating, and geography.

🔗 **[View live dashboard on Tableau Public](https://public.tableau.com/app/profile/hammad.farhat/viz/NetflixDashboard_17162196857520/Netflix)**

## Overview
Sourced and cleaned a dataset of 5,000+ Netflix titles, then analyzed content trends by genre, rating, and geography. Used Python for data preprocessing and Tableau for visualization and dashboard design.

## Key Findings
- **Documentaries** are the most common genre on the platform, making up **22%** of titles.
- **TV-MA** is the leading content rating, accounting for **35%** of titles — indicating a strong skew toward mature-audience content.
- Findings are relevant to content acquisition strategy and platform positioning decisions.

## Dashboard Features
- Interactive filters by country, genre, and year of release
- Visual breakdown of content distribution across ratings and genres

## Tools Used
`Python` (pandas) · `Tableau`

## Data Source
Netflix titles dataset (5,000+ entries)
